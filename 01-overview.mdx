# GitHub Overview

GitHub is a cloud platform for hosting and managing code using **Git**, a version control system. It lets individuals and teams track changes, collaborate, and ship software together.

## Core Concepts

- **Repository (repo)** — A project's folder, containing all files, history, and configuration.
- **Commit** — A saved snapshot of changes with a message describing what changed.
- **Branch** — A parallel line of development, used to work on features without affecting the main code.
- **Pull Request (PR)** — A request to merge changes from one branch into another, with room for review and discussion.
- **Issue** — A tracked task, bug, or idea tied to a repository.
- **Fork** — Your own copy of someone else's repository.

## What You Need

1. A [GitHub account](https://github.com/join)
2. Git installed locally ([git-scm.com](https://git-scm.com/downloads))
3. (Optional) A terminal or a Git GUI client (GitHub Desktop, VS Code, etc.)
