# Step 3: Clone and Local Setup

Cloning downloads a copy of a repository to your computer so you can work on it.

## 1. Copy the Repo URL

On the repo page, click **Code** → copy the **HTTPS** (or SSH) URL.

## 2. Clone It

```bash
git clone https://github.com/<username>/<repo-name>.git
cd <repo-name>
```

## 3. Configure Git (first time only)

```bash
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

## 4. Check Status

```bash
git status
```

This shows the current branch and any changed files.
